# 生命科学导论课程项目建议

## 项目定位
**主题**：生物学对计算科学的启发
**目标受众**：通识课（非计算机专业学生）
**核心**：展示"进化"的过程和效果

## 当前问题分析

### GA+BipedalWalker的问题
1. **效果不够惊艳**：173分勉强走几步，不够流畅
2. **训练时间长**：需要3-4小时才能看到效果
3. **不稳定**：运气成分大，多次训练结果差异大
4. **难以理解**：对生物专业学生来说，参数调优过于技术化

### 理想的Demo应该是：
1. ✅ **视觉效果好**：能明显看到"进化"过程
2. ✅ **结果惊艳**：最终表现很好，让人印象深刻
3. ✅ **易于理解**：不需要深入技术细节
4. ✅ **快速运行**：现场可以演示

## 🎯 方案建议

### 方案A：换简单环境 + GA（推荐）✅

#### 1. CartPole (倒立摆)
**描述**：平衡一根杆子
**优势**：
- 非常简单，GA 5-10分钟就能学会
- 效果明显：从摇晃到稳定
- 易于理解：就像人学骑自行车

**Demo效果**：
```
Gen 1:  杆子立刻倒下 ❌
Gen 10: 能坚持几秒 ⚡
Gen 50: 完全稳定 ✓✓✓
```

**实现**：
```python
ENV_NAME = "CartPole-v1"
HIDDEN_LAYERS = [16, 16]  # 很小的网络就够
GENERATIONS = 100
```

#### 2. MountainCar (爬山车)
**描述**：小车通过摇摆爬上山
**优势**：
- 直观展示"策略进化"
- 从随机到有规律的摇摆
- 视觉效果好

**Demo效果**：
```
Gen 1:  车乱动，到不了山顶 ❌
Gen 50: 学会摇摆，但还不够 ⚡
Gen 200: 完美爬到山顶 ✓✓✓
```

#### 3. LunarLander (月球着陆)
**描述**：火箭着陆
**优势**：
- 科幻感强，吸引眼球
- 从乱飞到精准着陆
- 中等难度，GA可以解决

**Demo效果**：
```
Gen 1:   坠毁 💥
Gen 100: 勉强着陆 🚀
Gen 300: 完美降落 ✅
```

### 方案B：用PPO + BipedalWalker（效果最好）✅

**为什么**：
- 2小时训练到280+分
- 走路非常流畅，效果惊艳
- 但不是"遗传算法"

**解决方案**：
```
"虽然我们最终用了更先进的强化学习方法，
但它的核心思想——试错和优化——
仍然源自生物进化的启发"
```

**代码**：
```python

model.learn(total_timesteps=2000000)
```
