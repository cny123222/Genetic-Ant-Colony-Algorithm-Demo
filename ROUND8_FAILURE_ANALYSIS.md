## 失败原因

### 1. 变异率过大破坏进化
**理论**：
- 适当变异 = 探索新区域
- 过度变异 = 破坏已有好策略

**证据**：
```
Gen 4-100 (96代): -18.85 → -17.56 
进步仅1.3分！几乎停滞！

对比Round 7前100代：
Gen 4-100: 大约从-20 → 80+
进步约100分！
```

**结论**: 0.35的变异率**太激进**，优秀个体的基因被大幅扰动，无法保持

### 2. 精英数太少（2个）
**问题**: 只保留2个精英，优秀策略容易丢失
**Round 7**: 3个精英，更稳定

### 3. 选择压力过小
**锦标赛4人** vs Round 7的5人
→ 选择不够严格，劣质个体存活太久

## 关键教训

### 教训1: 增量调整很重要
```
❌ 错误: 0.25 → 0.35 (+40%) 
✓ 正确: 0.25 → 0.28 (+12%)
```

**原因**: 
- 参数空间复杂，大幅改动容易适得其反
- 应该小步迭代，观察效果

```
0.20: 可能太保守

```
